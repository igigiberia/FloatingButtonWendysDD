# FloatingButtonWendysDD


Config parameters:

DefaultFilePath			დირექტორიის მისამართი, \ -> \\ გათვალისწინებით
ProcessName				პროცესის სახელი (.exe -ს გარეშე)
ButtonWidth				ღილაკის სიგანე (0 - ეკრანის სიგანე)
ButtonHeight			ღილაკის სიმაღლე (0 - ეკრანის სიმაღლე)
Left					ღილაკის X მდებარეობა (True - მარცხნივ, False - მარჯვნივ)
Top						ღილაკის Y მდებარეობა (True - ზემოთ, False - ქვემოთ)
